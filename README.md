# 👑 מיכאל המלך - תווים 🎹

אפליקציה צבעונית ושמחה לילד/ה בגיל 9-12 שמנגן/ת בפסנתר/קלידים:
מעלים קובץ שיר (MP3), והאפליקציה מזהה את התווים, מציגה אותם **בעברית**
(דו רה מי פה סול לה סי), מדגישה כל תו בזמן אמת תוך כדי האזנה,
מציירת מקלדת פסנתר, ומאפשרת אפילו לשחק "נגנו אחריי" עם המיקרופון.

זיהוי התווים מקובץ נעשה דרך **Klangio API** (מנוע זיהוי תווים מבוסס AI),
דרך ה-backend שלנו — כדי לא לחשוף את מפתח ה-API בדפדפן.

---

## 🚀 הרצה מהירה

צריך **Python 3.9+** מותקן.

```bash
# 1. נכנסים לתיקיית הפרויקט
cd "Michael app"

# 2. (מומלץ) יוצרים סביבה וירטואלית
python3 -m venv venv
source venv/bin/activate        # ב-Windows:  venv\Scripts\activate

# 3. מתקינים את הספריות
pip install -r requirements.txt

# 4. מגדירים את מפתח ה-API (ראו סעיף Klangio למטה)
cp .env.example .env
#   ועורכים את .env, מכניסים את KLANGIO_API_KEY

# 5. מריצים
python app.py
```

ואז פותחים בדפדפן: **http://localhost:5050** 🎉

> 💡 **רוצים לראות את האפליקציה עכשיו בלי מפתח?** פשוט הריצו `python app.py`
> ולחצו על הכפתור **"נגנו שיר דוגמה (כוכב קטן)"**. כל האפליקציה עובדת
> (כרטיסיות, מקלדת, הדגשה, מצב משחק) — בלי קובץ ובלי Klangio.

## 🔑 מפתח Klangio API

זיהוי תווים מקובץ אמיתי דורש מפתח API מ-Klangio:

1. נכנסים ל-https://klang.io/api וממלאים את טופס הבקשה ל-API (זה ה-API
   למפתחים — נפרד מהמנוי הרגיל של אפליקציות Klangio).
2. אחרי אישור, המפתח מופיע בלוח הבקרה: https://api-dashboard.klang.io
3. מעתיקים אותו לקובץ `.env`:
   ```
   KLANGIO_API_KEY=המפתח_שלכם_כאן
   KLANGIO_MODEL=piano
   ```

⚠️ **מגבלות חשובות לדעת מראש** (נכון להרשמה ל-API):

| חבילה | מחיר | אורך אודיו מקסימלי | בקשות |
|-------|------|--------------------|--------|
| Basic (חינם) | $0 | **15 שניות בלבד** | 50 סה"כ |
| Startup | $99/חודש | 300 שניות (5 דק') | 500 |
| Business | $499/חודש | 300 שניות | 3000 |

➡️ בחבילה **החינמית אפשר לנתח רק קטעים של עד 15 שניות**. לשיר שלם
צריך חבילה בתשלום. לכן מובנה בקוד **caching לפי תוכן הקובץ**: אם מעלים
שוב שיר שכבר נותח — התוצאה נטענת מהדאטהבייס בלי לקרוא (ולשלם) שוב ל-Klangio.

🔒 המפתח נטען **רק** ממשתנה סביבה (`.env`) ולעולם לא כתוב בקוד. קובץ
`.env` כבר נמצא ב-`.gitignore` כדי שלא יידחף בטעות ל-GitHub.

## ✨ מה יש באפליקציה

- 🎵 **העלאת MP3** (גרירה או בחירת קובץ) ושליחה לזיהוי בשרת.
- 🌈 **כרטיסיות תווים בעברית** — לכל תו צבע קבוע מצבעי הקשת
  (דו=אדום, רה=כתום, מי=צהוב, פה=ירוק, סול=תכלת, לה=כחול, סי=סגול).
- ⏱️ **הדגשה בזמן אמת** — התו הנוכחי קופץ ומאיר תוך כדי שמיעת השיר.
- 🎹 **מקלדת פסנתר גרפית** — המקש המתאים נצבע ו"נלחץ" עם כל תו.
- 🐢 **מצב "נגנו לי לאט יותר"** — מהירות רגיל / לאט / לאט מאוד (בלי לעוות את הצליל).
- 💾 **"השירים שלי"** — שירים שנותחו נשמרים בדאטהבייס ונפתחים מיד בלי ניתוח מחדש.
- 🎮 **מצב משחק "נגנו אחריי"** — הילד/ה מנגנ/ת במיקרופון והאפליקציה בודקת אם פגע/ה בתו (זיהוי בצד הדפדפן, משוב חיובי בלבד).
- 🌟 **שיר דוגמה מובנה** ("כוכב קטן") שעובד בלי מפתח ובלי קובץ.

האפליקציה מציינת זאת בעדינות למשתמש, ועדיף להעלות קטעים של מנגינה ברורה.
